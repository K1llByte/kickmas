# Merry Kickmas

**Merry Kickmas** is a fast-paced platformer keep-up game made for the **ScoreSpace Game Jam #37**.

**Play the game:** https://gh.killbyte.dev/kickmas/web

**Itch.io page:** https://mimiki-cafe.itch.io/merry-kickmas
## How to Play

- **Arrow keys / A & D:** Run left/right.
- **Space / Up / W:** Jump.
- **S / Down:** Get down faster.
- **Enter:** Kick the gift!

- **Goal:** Keep kicking without letting the gift hit the ground. Jump over chimneys and don't fall between buildings! The longer you go, the better you score. Can you become #1 on the leaderboard?
- Trouble controlling the ball? Try to hit it with the foot to send it forward, and with the back to send it upwards.
